Lab 1: The Artemis board

Setup and become familiar with the Arduino IDE and the Artemis board.

Posted by Zhihui Liu on Jan 31, 2023

Part1: Blink it up

The first part is straightforward. I followed the provided instruction, downloaded Arduino IDE, connected the Artemis board to the laptop via Type-C cable and ran the example code "Blink". As a result, the LED blinked at 1 Hz.

Part2: Serial Port

After uploading the example code "Serial" and opening the serial monitor, the board can read from the input and echo that to the laptop.

Part3: Analog Read

Firstly, I uploaded the example code "AnalogRead" and opened the serial monitor. At the beginning, the temperature was normal and the reading was around 32500. After pressing the temperature sensor for nearly one minute, the reading became around 32800.

Part4: Microphone Output

In this part, the highest frequency of the sound were output to the serial monitor. Initially the frequency was around 500 Hz. This figure then rose up to around 3000 when whistling.

Additional Task: "A" Note Detection

To turn the LED on when "A" note (440 Hz) is played and off otherwise, I edited the code from the last example "MicrophoneOutput":

1. In the 'setup()', I initialized the pin LED_BUILTIN as an output.
pinMode(LED_BUILTIN, OUTPUT);

2. In the printLoudest() function in the loop, I changed the 'if' statement.
if(ui32LoudestFrequency>430 && ui32LoudestFrequency<450){
digitalWrite(LED_BUILTIN, HIGH);
}
else{
digitalWrite(LED_BUILTIN, LOW);
}
